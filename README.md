## Getting Started

### Prerequisites

- For Starting

```sh
npm run start
```

- For Building

```sh
npm run build
```

### Installation

1. Clone the repo

```sh
git clone https://github.com/guibais/Gatsby-Portfolio.git
```

2. Install NPM packages

```sh
npm install
```
